Goblin Browser RPG
2020

-

A vanilla javaScript turn based RPG.  Each action of the player sent an API call to a server with the current state of the player and returned back the results of that action and what the player's options are. It was a challenge to handle state without something like React. An RPG is a great way to make state as complicated as possible, and come up with a design pattern that allowed the game to continue to grow.

mapChat
2019

-

mapChat utilized Google Firebase to create users and allow them to pin messages on a map and share them. Users could then see and share messages from users nearby and take advantage of various overlays to see the coorelation between time, weather, temperature, and user activity. To make the app performant, messages were stored on a noSQL FireStore database using geohashes to turn what would've been a very complicated lat-long lookup into a 1D range of results.
